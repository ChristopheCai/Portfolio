Treasure Hunter

This game is a project of a coin runner where the player needs to search for treasure (coins) and escape from monsters in a horror-themed level.

Play the game on itch.io

Introduction

I made this project for my school project. We needed to do a Coin Runner on Unity. All the tools were free to use.
I spent 2 weeks on this project.
I used the following tools to make it :

  • - Unity
  • - Unity Assets (AI Navigation, Cinemachine, Input System)
  • - Sounds from Pixabay or Freesound
  • - Models and textures from Sketchfab
  • - Animation from Mixamo
  • - Background from Dall-E

Originality

This Coin Runner stands out by the atmosphere of horror game that isn't common in that kind of game.
The fact of having monsters keeping this too and in a original level design.
Each monsters have different behaviors depending on the level.

Character

I've started to put a lot of my time for character controller to make the player feels the most confortable while using it. So I choose to use a Character Controller instead of a RigidBody because it offers more control on movement.

Immersion

To immerse the player, audio and visual are important and being implemented.
I used a lot of time to consider those to match with the horror atmosphere I wanted to create.
I added Cinemachine to control the first and third person view in order to give the player more immersion on gameplay.
To make it even more immersive, I added a lots of animations for the player but also the objects interactable, the monsters, and others...

Difficulties & Solutions

  • Smooth Movement

    Make the movement looks fitting the context.

  • Solution

    Using Input System & Cinemachine with customed movement gestion.

  • Animation

    The animation used are always only from one specific rotation.

  • Solution

    Using Blending Tree to make the animation into all direction and their transition smoother.

  • Audio & Visual

    Make the Audio & Visual fit the horror atmosphere wanted. With coherence between them.

    Simply took a lot of time to search for free assets available from various sources.
